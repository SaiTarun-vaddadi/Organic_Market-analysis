# Organic_Market-analysis
A data analysis project focused on identifying best-selling organic products through survey-based data collection, structured dataset preparation, exploratory data analysis, and visualization using Python.
This project analyzes product demand and sales trends across organic retail stores using primary survey data. The objective was to identify best-selling organic products and compare store-level performance using structured data analysis.

Key Objectives:
1.Collect real-world data from organic retail stores
2.Structure and clean raw survey data in Excel
3.Perform Exploratory Data Analysis (EDA) using Python
4.Identify high-demand product categories
5.Generate comparative insights using data querying and visualizations

Tools & Technologies:
Python
Pandas
Matplotlib
Excel
Jupyter Notebook

Key Insights:
Identified top-performing organic product categories
Analyzed frequency distribution of products across stores
Compared demand patterns across different retail locations
